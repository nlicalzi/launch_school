# (Local) Variable Scope
- How local variables interact with method invocations with blocks
- How local variables interact with method definitions
- Types of variables (constant, global, class, local)
  - Constant variables: global scope, (should be) immutable
    - Ex.: MY_CONSTANT = 'A constant variable'
  - Global variables: global scope, mutable (try not to use)
    - Ex.: $var = 'A global variable'
  - Class variables: local to the class/its instances, mutable
    - Ex.: @@instances = 'A class/instance variable'
